I recommend a video translation and dubbing tool called AMDLocalDub, also known as AMD Offline Dubbing. This software can automatically convert foreign language videos into Chinese dubbed versions, and the entire process runs entirely on your own computer — no need to upload videos to any cloud server. It is specifically optimized for AMD graphics cards: speech recognition runs on AMD GPU via whisper.cpp's Vulkan backend, and video encoding also uses AMD's AMF hardware acceleration, so you don't need an NVIDIA GPU for smooth performance. For translation, it supports three engine options — if you load a local model with LM Studio, you can achieve fully offline use with no internet connection required. Operation is simple: double-click the startup script, open the browser interface, enter or drag in a video file, select the translation engine, and click start — it automatically completes speech recognition, translation, dubbing, and video composition. It supports common formats like MP4 and MKV, and outputs finished videos with hardcoded subtitles in white text on a black background. If you have an AMD graphics card and are looking for a fully local video translation and dubbing solution, this is worth a try.
