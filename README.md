# ⚡ AMDLocalDub — AMD离线配音

> **纯本地运行 · AMD GPU 加速 · 视频翻译配音工具**

---

## 这软件能干什么？

把外文视频自动转成**中文配音版**（或其他语言），一切都**在你自己的电脑上完成**，不需要任何云端服务。

```
输入: 英文视频 → ASR语音识别 → 翻译 → TTS配音 → 视频合成 → 输出: 中文配音版
```

---

## 凭什么用 AMD 显卡？

本软件针对 **AMD Radeon RX 9070 XT** 做了完整适配：

| 环节 | 用什么加速 | 显卡类型 |
|------|-----------|---------|
| 语音识别 | whisper.cpp **Vulkan** 后端 | ✅ **AMD GPU** |
| 视频编码 | ffmpeg **AMF** (h264\_amf) | ✅ **AMD GPU** |
| 翻译引擎 | CPU（API 调用，不吃显卡） | — |
| TTS 配音 | CPU（Edge-TTS） | — |

> 全程不依赖 CUDA / NVIDIA，**AMD 显卡用户也能获得 GPU 加速**。

---

## 凭什么说完全本地？

| 功能 | 是否联网 | 说明 |
|------|---------|------|
| 语音识别 | ❌ **纯本地** | whisper.cpp，模型在本地加载 |
| 翻译 | ⚠️ **可选** | 支持三种引擎： |
| | | **SiliconFlow API**（需联网） |
| | | **DeepSeek API**（需联网） |
| | | **LM Studio 本地模型**（纯本地，不联网） |
| TTS 配音 | ❌ **纯本地** | Edge-TTS 使用系统接口 |
| 视频合成 | ❌ **纯本地** | ffmpeg 本地编码 |
| 全部过程 | ❌ **无云端中转** | 视频不上传任何服务器 |

> 搭配 LM Studio 本地模型使用翻译，可实现**完全断网运行**。

## 快速开始

```bash

# 2. 浏览器打开
http://127.0.0.1:7860

# 3. 操作
#    - 在文本框输入视频路径（推荐，保留原目录）

```

## 主要功能

- **语音识别**: whisper.cpp + Vulkan GPU 加速（AMD 显卡），99 种语言
- **翻译引擎**: SiliconFlow / LM Studio 本地 / DeepSeek API 自由切换
- **智能拆分**: 按语义边界分段，合并短句，拆分超长句
- **TTS 配音**: Edge-TTS，23 种语言，50+ 种音色可选
- **视频合成**: AMD AMF GPU 编码，字幕硬烧录（白字黑底）
- **时间轴对齐**: 自动拉伸 TTS 音频匹配字幕时间戳
- **批量处理**: 多文件排队，依次处理
- **纯本地运行**: 配合 LM Studio 可实现完全离线
